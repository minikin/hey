## Requirements

- iOS 9.0+ / macOS 10.13+
- Xcode 9.3+
- Swift 4.1+
- CocoaPods 1.5.0 +

## Installation

```bash
git clone https://github.com/minikin/hey.git && cd hey && make setup
```

## Running the project

From Xcode you can run the app `Hey.xcworkspace`.
